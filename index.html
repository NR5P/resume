Recent Personal Project

Home Automation Timer

github-https://github.com/NR5P/complete_plant_automation

techologies - C++, NodeJS, Express

This project is partly embedded raspberry pi and a webapp. This project started when I found there were not many options for automated irrigation. This runs a server on the raspberry pi which serves a node web app. The web allows control of the settings for the automation controller which is written in C++. The raspberry pi's pins run a module that has solid state switches to control 24dc timers or 110v a/c accessories. Originally this was built with irrigation timers in mind but I'm adding the ability to add heaters, grow lights, and various other options that would be automated on a small farm or homestead. I'm using this project as a way to get experience in the Rust language and have begun to rewrite the C++ part in Rust.
